# Fitness Coach Pro - Stage 6 Report

## Этап 6: PDF и экспорт

### Что добавлено

1. Печатная версия тренировочного плана:
   - новая страница `/admin/plans/[id]/print`
   - чистый вид без форм редактирования
   - фото упражнений
   - техника выполнения
   - частые ошибки
   - ограничения
   - блок отчёта после тренировки

2. PDF через браузер:
   - кнопка `Скачать PDF`
   - используется стандартная печать браузера
   - в окне печати нужно выбрать `Сохранить как PDF`
   - это надёжно для MVP и не требует тяжёлых PDF-библиотек

3. WhatsApp-экспорт:
   - отдельный блок текста для WhatsApp
   - кнопка копирования текста
   - текст включает клиента, цель, период, дни тренировок и отчёт после тренировки

4. Улучшена страница редактирования плана:
   - добавлена кнопка `PDF / печать`
   - добавлена ссылка на экспорт рядом с WhatsApp-блоком

5. CSS для печати:
   - скрытие служебных кнопок при печати
   - A4-формат
   - защита карточек упражнений от разрыва между страницами

### Проверка

1. Запустить проект:

```bash
npm install
npm run dev
```

2. Открыть:

```text
/admin/login
```

3. Войти в админку.

4. Открыть тренировочный план:

```text
/admin/plans/[id]
```

5. Нажать `PDF / печать`.

6. На странице экспорта нажать `Скачать PDF`.

7. В окне печати выбрать `Сохранить как PDF`.

8. Проверить кнопку `Скопировать WhatsApp`.

### Команды контроля качества

```bash
npm run build
```

В этом проекте `npm run build` выполняет строгую TypeScript-проверку через `tsc --noEmit`.

Для реальной production-сборки Next.js можно отдельно запустить:

```bash
npm run next:build
```
