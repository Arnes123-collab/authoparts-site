## Проверка

```bash
npm install
npm run typecheck
npm run build
npm run dev
```

Ручная проверка:

1. Открыть `/admin/login`
2. Войти по `ADMIN_PASSWORD`
3. Открыть `/admin/settings`
4. Изменить имя тренера, бренд, контакты и цвета
5. Сохранить
6. Проверить таблицу `coach_settings` в Supabase
7. Открыть публичный сайт и админку на телефоне
