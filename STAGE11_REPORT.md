# Fitness Coach Pro - Stage 11

Этап 11 добавляет расширенную CMS упражнений для коммерческого шаблона Fitness Coach Pro.

## Что добавлено

- Расширенная карточка упражнения
- Категория и подкатегория
- Мышечная группа
- Оборудование
- Уровень сложности
- Спорт-теги
- Замена упражнения
- Заметки тренера
- Включение / выключение упражнения
- Поиск и фильтры в `/admin/exercises`
- API фильтрации `/api/exercises`
- API обновления `/api/exercises/[id]`
- SQL `supabase/011_stage11_exercise_cms.sql`

## Проверка

1. Выполнить SQL `supabase/011_stage11_exercise_cms.sql` в Supabase SQL Editor.
2. Запустить `npm install` при первом запуске.
3. Запустить `npm run dev`.
4. Открыть `/admin/login`.
5. Перейти в `/admin/exercises`.
6. Добавить упражнение с категорией, подкатегорией, тегами и заменой.
7. Проверить фильтры: поиск, категория, мышца, активность.
8. Нажать `Выключить` у упражнения.
9. Убедиться, что скрытое упражнение не предлагается в конструкторе тренировочного плана.
10. Запустить `npm run typecheck`.

## Важно

Скрытие упражнения не удаляет его из базы. Оно только убирает упражнение из активной библиотеки для новых тренировочных планов.
